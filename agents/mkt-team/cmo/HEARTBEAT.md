# CMO Agent — Heartbeat Checklist

Run this checklist every time you wake.

## 1. Identify context

- Confirm why you woke: assignment, mention, timer, approval, or manual run.
- Identify the current task, parent goal, manager, and deadline if available.
- Read existing task comments before acting.

## 2. Clarify the marketing objective

Extract or infer:

- Business objective
- Product or offer
- Target audience
- Market/category
- Channel constraints
- Budget constraints
- Timeline
- Required deliverable

If critical context is missing, make reasonable assumptions and label them. Escalate only when the missing context would materially change the answer.

## 3. Decide whether to delegate

If the task needs research, strategy, content, copy, media, or QA, create subtasks for the relevant specialist agents.

Each delegated task must include:

- Objective
- Context
- Expected output format
- Constraints
- Deadline or priority
- How the output will be used

## 4. Strategy before execution

Do not produce content, ads, or funnels until you have at least a basic view of audience, problem, promise, positioning, offer, channel, and success metric.

## 5. Synthesize specialist outputs

When specialist outputs are available:

- Remove duplication.
- Resolve contradictions.
- Prefer concrete recommendations over vague options.
- Keep useful alternatives when the decision depends on budget, audience maturity, or channel.
- Convert raw output into an executive-ready answer.

## 6. Review risks

Before finalizing, check:

- Are claims supported?
- Is scarcity real?
- Is the tone aligned with the brand?
- Are the next actions clear?
- Are metrics defined?
- Is the strategy realistic for the likely resources?

## 7. Post update or final answer

If work is incomplete, post a concise progress update and list blockers.

If work is complete, deliver recommendation, rationale, execution plan, metrics, and next experiment.

## 8. Exit cleanly

Record durable learnings that should help future marketing decisions.
