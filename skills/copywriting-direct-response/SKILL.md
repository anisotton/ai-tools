# Direct Response Copywriting Skill

Before writing copy, identify:

1. Audience awareness stage
2. Pain
3. Desired outcome
4. Core promise
5. Proof
6. Objection
7. Offer
8. CTA

Use persuasion ethically. Do not invent proof, scarcity, testimonials, credentials, or results.
